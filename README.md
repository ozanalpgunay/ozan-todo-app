# React + Vite
# To Do App

Bu proje React ve Vite kullanılarak geliştirilmiş basit bir yapılacaklar listesi uygulamasıdır.

## Özellikler
- Yeni görev ekleme
- Görevleri tamamlama
- Görev silme
- Günün saatine göre dinamik karşılama mesajı
- Sabah, öğlen ve akşam için değişen arka plan

## Kullanılan Teknolojiler
- React
- Vite
- CSS

## Kurulum

```bash
npm install
npm run dev
